## Getting Started

- Review `package.json`
- Run `yarn install`
- To do work, run `yarn dev`

**Runs on http://localhost:9000**

**Testing out the API**

````bash
curl --location 'http://localhost:9000/stock/' \
--header 'X-Api-Key: SuperSecretApiKey123'```
````
